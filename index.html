A fully functional calculator built with HTML, CSS, and JavaScript. This project focuses on DOM manipulation, event handling, and basic arithmetic operations. It includes support for addition, subtraction, multiplication, and division, with a clean and responsive design.

Sign-up Form

A responsive and user-friendly sign-up form with client-side validation. This project emphasizes form design, accessibility, and JavaScript validation for fields like email, password, and phone number. It also includes error messages for invalid inputs.

Etch-a-Sketch

An interactive drawing app inspired by the classic Etch-a-Sketch toy. Users can draw on a grid by hovering over cells, with options to adjust the grid size and clear the canvas. This project highlights DOM manipulation, event listeners, and CSS grid.

Rock Paper Scissors

A simple yet engaging game of Rock, Paper, Scissors against the computer. This project introduces basic JavaScript logic, random number generation, and user interaction. It includes a score tracker and a clean UI for a seamless gaming experience.
